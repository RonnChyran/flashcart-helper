Guide for eNDryptS
If you are seeing this and you have not used FlashcartHelper to download eNDecryptS, disregard this information.
This guide was made by ron975.

i)Only TTmenu or YSMenu users need to decrypt some roms.

1)Go into the eNDcryptS folder
2)Put games in the eNDcryptS folder
3)If you are on Windows Vista or Windows 7, Start eNDecryptS as administrator 
4)Press 1 to Encrypt/Decrypt roms
5)Check to see if your rom is decrypted
6)Press any key to go back to the main menu
7)Press 0 to exit
8)Roms are now decrypted
